Wavelogger

This is for managing and understanding your open wave height logger data

Requirements

  • R v4+
  • dplyr
  • readr
  • stringr
  • ggplot2
  • oceanwaves
  • oce
  • owhlR from github
  • mesowest from github

Installation

remotes::install_github("rfrancolini/wavelogger")
